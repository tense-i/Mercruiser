# Mercruiser 重构版业务 PRD

## 文档信息
- 文档名：Mercruiser 重构版业务 PRD
- 版本：v2.0
- 状态：Draft
- 目标版本：Web Studio 重构版
- 适用范围：系列化 AI 短剧 / 漫剧生产

## 1. 文档目的

本 PRD 用于定义 Mercruiser 下一阶段的核心业务模型、生产工作流和产品范围。

本次重构的核心目标不是简单优化页面，而是将产品从“阶段式工作台”升级为“系列治理 + 单集工业化生产线”的双层体系。

## 2. 产品定义

Mercruiser 是一个面向 AI 短剧 / 漫剧生产的 Studio。

其本质不是单次生成工具，而是一个持续生产系统，包含：

- 系列治理层
- 单集生产线层
- Agent 编排层
- 任务与系统治理层

## 3. 产品定位

### 3.1 系列原生
产品以“系列”为顶层对象，以“集数”为交付单元，以“章节 / 镜头”为生产颗粒度。

### 3.2 工业化生产线
单集内部不是一个大而全工作台，而是一条明确生产线：

1. 概览
2. 剧本
3. 主体
4. 分镜
5. 故事板
6. 成片

### 3.3 Agent 编排驱动
Agent 不只是聊天工具，而是生产协调器，负责：

- 当前状态识别
- 缺口诊断
- 下一步建议
- 任务编排
- 失败恢复建议
- 质量提示

### 3.4 资产沉淀与复用
每一集生产不是孤立行为，高价值资产、风格规则、提示词经验必须沉淀回系列层。

## 4. 核心业务结构

### 4.1 系列层
负责：

- 系列基础信息
- 世界观 / 风格 / 导演规则
- 系列共享主体资产
- 系列级模型 / Prompt / Agent 策略
- 集数列表与总进度
- 系列级风险管理

### 4.2 集数层
负责：

- 单集源内容
- 单集章节结构
- 单集镜头结构
- 单集故事板
- 单集成片
- 单集导出结果

### 4.3 章节层
章节是剧本工位的工作单位，负责：

- 承载文本内容
- 控制字数与节奏
- 作为后续主体提取、分镜拆解的输入源

### 4.4 镜头层
镜头是分镜工位的工作单位，负责：

- 场景
- Prompt (提示词)
- 景别与构图
- 运镜 (Camera Motion)
- 光影
- 对白
- 音效
- 时长 (支持音频前置驱动的自动计算)
- 分镜描述
- Take (候选条) 管理
- 连戏状态 (Continuity Status)

### 4.5 故事板层
故事板是镜头的多模态装配层，负责：

- 镜头图
- 参考主体
- 分镜素材
- 镜头视频候选
- 选定版本

### 4.6 Agent 编排层
横跨全局，负责：

- 阶段推进
- 前置条件校验
- 推荐动作
- 批量动作触发
- 失败恢复建议

## 5. 标准业务主线

### 5.1 系列建立
1. 创建系列
2. 配置系列设定
3. 配置系列策略
4. 导入或创建集数

### 5.2 单集生产
1. 进入单集
2. 在“剧本”页编辑章节内容
3. 提取主体
4. 生成主体图并沉淀主资产
5. 生成分镜镜头表
6. 进入故事板生成镜头图/视频
7. 进入成片页剪辑与确认
8. 导出结果

### 5.3 反哺系列
1. 将高价值主体升级为系列共享资产
2. 将有效策略沉淀为系列 Prompt / Agent 策略
3. 将失败经验沉淀为恢复建议

## 6. 用户角色

### 6.1 系列制片 / 内容统筹
关注系列总进度、设定统一性、资源复用率和交付节奏。

### 6.2 集数导演 / 创作执行者
关注单集从文本到成片的推进效率。

### 6.3 Prompt / 流程调优操作者
关注模型、提示词、Agent 策略和稳定性。

### 6.4 系统配置管理员
关注供应商、模型、请求、数据与运行环境。

## 7. 功能范围

### 7.1 工作区首页
必须支持：

- 系列列表
- 系列状态摘要
- 新建系列
- 导入创建
- 快速进入最近生产中的系列

### 7.2 系列详情
必须支持以下子区：

- 总览
- 系列设定
- 集数管理
- 共享主体资产
- 系列策略配置

### 7.3 单集执行页
必须重构为以下固定页签：

- 概览
- 剧本
- 主体
- 分镜
- 故事板
- 成片

### 7.4 剧本工位
必须支持：

- 左侧章节列表
- 中部章节文本编辑
- 右侧全局设定侧边栏 (包含视频比例、创作模式、风格参考等)
- 音频前置生成 (Audio-First Workflow)：先生成全局旁白/对白音频
- 一键拆解分镜 (Breakdown to Shots) 与资产提取
- 新增章节
- 章节顺序管理

### 7.5 主体工位
必须支持：

- 角色 / 场景 / 道具分类
- 主体卡片列表 (支持绑定 Voice 配音选择)
- 主体描述编辑
- 资产状态管理 (Stateful Assets)：支持同一角色的多状态管理（如默认、战损、特定服装）
- 资产图片版本管理 (Asset Versioning)：支持多次生成资产图片，保留历史版本，并可随时切换主版本
- 右侧资产生成专属工作区 (支持提示词编辑、模型选择、参考图上传及生成素材池)
- 主版本选择
- 面部特征锁定 (Face Locking) 及一致性视觉标识
- 升级为系列共享资产

### 7.6 智能分镜表工位 (Shot List)
必须支持：

- 垂直列表布局，多列网格数据结构展示
- 镜头表结构化编辑 (包含场景、景别、光影、Prompt 提示词、对白、时长等)
- 运镜控制 (Camera Motion，如推拉摇移)
- 新增 / 删除 / 复制 / 重排镜头
- 分镜表导入导出

### 7.7 故事板工位 (Storyboard)
必须支持专业级视觉生成工作台布局：

- 左上监视器 (Monitor)：超大画幅预览，支持字幕编辑与音频试听
- 左下镜头时间线 (Timeline)：横向滚动缩略图列表，支持快速切换与状态查看，集成 AI 连戏检测警告 (Continuity Check)
- 右侧生成控制台：包含图/视频切换、当前镜头 Prompt 编辑、模型选择
- 参考主体 (Reference Subjects) 快速关联 (支持选择特定资产状态)
- Take (条) 与版本管理：支持图片与视频的多次生成抽卡，保留独立的图片候选池与视频候选池，并支持主版本选定 (Mark as Selected)
- 批量生成图片 / 视频

### 7.8 成片工位
必须支持：

- 多轨非线性编辑 (NLE) 时间线
- 视频轨、对白轨 (Dialogue)、音效/BGM轨对齐与装配
- 顺序调整与片段裁剪 (Scissors)
- 分辨率/帧率设置与全局预览
- 片段替换
- 导出前确认

### 7.9 任务中心
必须支持：

- 查看任务
- 筛选任务
- 查看失败原因
- 重试
- 回溯到对应工位

### 7.10 设置中心
必须支持：

- 语言
- 供应商
- 模型
- Prompt
- Skill
- Memory
- Agent
- 请求
- 文件
- 数据

## 9. 关键业务规则

1. 系列级规则默认继承到单集。
2. 单集可临时覆盖，但不能静默污染系列级规则。
3. 主体资产分为“本集主体”和“系列共享主体”。
4. 系列共享主体必须具备主版本。
5. 已锁定主体、分镜、视频应有明确视觉标记。
6. 故事板与成片只能使用已选版本 (Selected Take) 进入下游。
7. 审校未通过不得进入导出。
8. 开启“面部锁定 (Face Locked)”的角色资产，在后续分镜生成中必须强制应用一致性约束（如垫图、LoRA 或 Reference ID）。
9. 音频前置模式下，镜头时长由绑定的对白/音效长度自动锁定，手动修改需解除绑定。
10. 数据飞轮与私有化模型：当系列累积足够多的 Selected 优质镜头后，系统可提示用户训练该系列专属的风格微调模型 (Custom LoRA)。

## 10. MVP 定义

- 系列管理
- 单集六工位主线
- 共享主体资产
- 分镜表
- 故事板生成
- 成片装配
- 任务恢复
- 设置中心基础治理

### P1
- 多成员协作
- 镜头批量策略模板
- 更强的审校系统
- 导出历史增强

### P2
- 运营分析
- 多角色审批
- 更深的后期能力

## 11. 成功指标

### 11.1 业务指标
- 单集完成时间降低
- 共享资产复用率提升
- 分镜返工率下降
- 视频候选有效率提升

### 11.2 质量指标
- 角色一致性提升
- 镜头节奏稳定性提升
- 失败任务恢复时间缩短

### 11.3 系统指标
- 关键页操作成功率
- 任务执行成功率
- 页面响应速度
- 导出成功率
